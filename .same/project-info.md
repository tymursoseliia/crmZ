## Описание проекта

Это Next.js приложение для генерации автомобильных документов:

### Основные функции:
1. **Генерация актов осмотра** - TÜV Rheinland и DEKRA
2. **Создание талонов очереди** - Электронная очередь
3. **Управление компаниями** - База данных компаний
4. **Учёт финансов** - Финансовая отчётность

### Структура приложения:

#### Страницы:
- `/` - Главная страница с выбором типа документа
- `/login` - Страница входа
- `/companies` - Управление компаниями
- `/finances` - Финансовый учёт
- `/ticket` - Генерация талонов

#### API endpoints:
- `/api/auth/login` - Авторизация
- `/api/auth/logout` - Выход
- `/api/generate-pdf` - Генерация PDF документов
- `/api/generate-qr` - Генерация QR-кодов
- `/api/generate-ticket` - Генерация талонов
- `/api/convert-pdf` - Конвертация документов

### Технологии:
- Next.js 15.5.9 с App Router
- TypeScript
- Tailwind CSS
- shadcn/ui компоненты
- docxtemplater для работы с DOCX шаблонами
- qrcode для генерации QR-кодов
