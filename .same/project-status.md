# Статус проекта

## Информация о проекте
- **Название:** Империя СКАМА - Генератор автодокументов
- **Тип:** Next.js 15 + TypeScript + Supabase
- **Репозиторий:** https://github.com/gennadiyyasnov-maker/skamsite

## Основные возможности
1. **Генерация документов:**
   - TÜV Rheinland - акты осмотра автомобилей
   - DEKRA - акты осмотра автомобилей
   - Талоны очереди - электронная очередь

2. **Финансовый модуль:**
   - Управление операциями (растаможка, добив)
   - Учет расходов
   - Управление сотрудниками и командами
   - Автоматический расчет зарплат
   - Аналитика и отчеты

3. **Управление компаниями:**
   - База данных компаний
   - Документы компаний
