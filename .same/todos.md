## 📊 Что это за проект

Это **полноценная бизнес-система управления автобизнесом** под названием "Империя СКАМА" - веб-приложение для автоматизации документооборота, финансов и CRM.

---

## 🎯 Основной функционал

### 1. 📄 Генерация автодокументов
- **TÜV Rheinland акты осмотра** - проверка лакокрасочного покрытия автомобилей
- **DEKRA акты осмотра** - альтернативная форма проверки авто
- **Талоны электронной очереди** - система записи клиентов
- Генерация происходит через DOCX шаблоны с автоматической подстановкой данных

### 2. 💰 Финансовый модуль (детальная система)
**Структура команд:**
- Команда Вохи (менеджеры + тимлид Воха)
- Команда Зета (менеджеры + тимлид Зет)
- Офис (Жигуль, Клыча, клоузеры)
- IT отдел (разработчики с фикс. ставкой)
- Особые роли (Звук - 2% от общей прибыли, Фотошоп)

**Функции:**
- Учет операций: растаможка и добив автомобилей
- 4 типа расходов: персональные, технические, постоянные, общие
- Автоматический расчет зарплат по сложным формулам
- Минимальная ставка для тимлидов (350 USDT)
- Аналитика, графики, экспорт в Excel

### 3. 📊 CRM - Воронка продаж
- Отслеживание лидов по стадиям
- История всех изменений статусов
- Визуализация воронки с конверсией
- Анализ: откуда лиды переходят в "Потеряно"
- Статистика по командам
- Real-time обновления через Supabase

### 4. 🏢 Управление компаниями
- База данных из 8 компаний (Автопул, Автотрейдинвест, БТАТОРГ и др.)
- Хранение документов: сертификаты, реестры, договоры
- Карточки организаций

## 🛠️ Технологический стек

- **Frontend:** Next.js 15 (App Router), React 18, TypeScript
- **Styling:** Tailwind CSS + shadcn/ui компоненты
- **База данных:** Supabase (PostgreSQL) + localStorage для оффлайн режима
- **Документы:** docxtemplater (Word шаблоны), qrcode (QR генерация)
- **Графики:** Recharts
- **Export:** xlsx (Excel файлы), file-saver

## 🚀 Ключевые особенности архитектуры

1. **Динамические импорты** - оптимизация загрузки компонентов
2. **Context API** - глобальное состояние для финансов, CRM, телеграм данных
3. **Middleware** - проверка авторизации на уровне роутера
4. **API Routes** - серверная генерация PDF, QR кодов
5. **Real-time синхронизация** - через Supabase subscriptions
6. **Двойное хранение** - Supabase + localStorage (fallback)

## 📈 Бизнес-логика

**Расчет зарплат:**
- Менеджеры: % от растаможек/добивов (не меньше минималки)
- Клоузеры: 5% от добивов
- Тимлиды: MAX(350 USDT, 5% от прибыли команды)
- Звук: 2% от общей прибыли обеих команд
- IT отдел: фикс. ставка (делится 50/50 между командами)

**Расходы:**
- Персональные → относятся к команде
- Остальные → делятся 50/50 между командами
