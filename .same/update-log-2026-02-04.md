# 📊 Обновление: Аналитика CRM
**Дата:** 04.02.2026

## 🎯 Что добавлено

### Полная система аналитики воронки продаж

Теперь CRM включает продвинутую аналитику с отслеживанием движения лидов по всем этапам воронки продаж.

## ✨ Новые возможности

### 1. История изменений статусов
- ✅ Автоматическое отслеживание всех изменений статуса лида
- ✅ Запись: дата/время, предыдущий статус → новый статус, кто изменил
- ✅ Используется для детальной аналитики

### 2. Воронка продаж
- 📊 Визуальное представление количества лидов на каждом этапе
- 📈 Конверсия между этапами (зеленый >70%, желтый 50-70%, красный <50%)
- 💰 Отображение сумм по каждому этапу
- 🎨 Красивая градиентная визуализация

### 3. Анализ срезов (главная фича!)
**Решает вашу задачу:** показывает откуда именно лиды переходят в "Потеряно"

Пример отчета:
```
Откуда теряются лиды:
├─ Оплата за растаможку → 5 лидов (50% от всех потерь)
├─ Оплата за авто → 3 лида (30% от всех потерь)
└─ Дал реквизиты → 2 лида (20% от всех потерь)
```

Это помогает:
- 🎯 Выявить проблемные этапы воронки
- 💡 Понять где клиенты отваливаются
- 📊 Принимать решения на основе данных
- ⚡ Оптимизировать процессы

### 4. Среднее время на этапах
- ⏱️ Показывает сколько дней в среднем лид проводит на каждом этапе
- 🔍 Помогает найти узкие места
- 📅 Планировать сроки закрытия сделок

### 5. Общая статистика
Карточки с ключевыми метриками:
- 👥 **Всего лидов** - общее количество
- ✅ **Завершено** - успешные сделки + % конверсии
- ❌ **Потеряно** - срезы + % потерь
- 💵 **Выручка** - реальная + потенциальная

### 6. Статистика по командам
- Команда Вохи: всего / завершено / потеряно
- Команда Зета: всего / завершено / потеряно
- Сравнение эффективности

## 📖 Как использовать

### Быстрый старт:
1. Зайдите в раздел **CRM**
2. Переключитесь на вкладку **"Аналитика"**
3. Изучите метрики и графики

### Для новых данных:
- История автоматически записывается при изменении статуса
- Просто работайте с лидами как обычно

### Для существующих лидов:
- Запустите миграцию из `.same/migration-add-stage-history.sql`
- Или история начнет накапливаться с текущего момента

## 🎓 Примеры использования

### Кейс 1: Выявление проблем
```
Проблема: Низкая конверсия
Аналитика показала: 60% срезов на "Оплата за растаможку"
Решение: Пересмотреть ценообразование / улучшить коммуникацию
```

### Кейс 2: Оптимизация процессов
```
Данные: Среднее время на "Дал реквизиты" = 7 дней
Анализ: Слишком долго готовятся документы
Действие: Автоматизировать или делегировать
```

### Кейс 3: Сравнение команд
```
Команда Вохи: 80% конверсия, 12 дней до закрытия
Команда Зета: 60% конверсия, 18 дней до закрытия
Решение: Обмен опытом между командами
```
