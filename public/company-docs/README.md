## Инструкция

1. Загрузите PDF файлы с соответствующими названиями в эту папку
2. Убедитесь, что названия файлов точно совпадают с указанными выше
3. Документы будут доступны для скачивания на странице "Компании"

**Примечание:** Если документ не загружен, пользователи увидят ошибку при попытке скачивания.
